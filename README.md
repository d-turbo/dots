# Dos' dots

I use these files to make my Linux machine bearable to use.

## How to use

Run `make`. This will create symbolic links the config files.
Use `make suckless PROG=<program>` for suckless programs.
If in case of errors in patching, you should edit it first in your `/tmp` directory.
Then, run `make resume PROG=<program>` to continue. ~~Annoying, I know.~~
